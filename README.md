## Version

Current app version is on v1.0.

## Usage

```
# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start

# Initialize mongodb
mongod
```

## Endpoints

```
# Routes
ROOT          {{url}}/
POST          {{url}}/api/users
GET           {{url}}/api/users
GET BY ID     {{url}}/api/users/:userId
PUT           {{url}}/api/users/:userId
DELETE        {{url}}/api/users/:userId

```
