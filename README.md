# Codebase-Restify

A back end project template using Restify as a Node.js framework, Redis for data caching, and Apache Kafka as a distributed event streaming platform.

## Usage

```
# Install dependencies
npm install

# Run in development
npm run dev

# Run in production
npm start

# Unit / Integration testing
npm run test
npm run cover // To create a coverage report

# Check linting
npm run lint
npm run lint-fix // To fix linting problem

# Initialize MongoDB
./mongod

# Initialize Redis
./redis-server
```
